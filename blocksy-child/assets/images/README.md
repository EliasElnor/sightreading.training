# PianoMode Sight Reading Training - Assets

## Logo PianoMode

Pour que l'application fonctionne correctement, vous devez placer le logo PianoMode dans ce dossier :

**Fichier requis :** `Logo-def_NOIR.png`

### Instructions :

1. Téléchargez le logo PianoMode (format PNG, fond transparent)
2. Placez-le dans ce dossier : `/blocksy-child/assets/images/`
3. Assurez-vous que le fichier s'appelle exactement : `Logo-def_NOIR.png`

### Spécifications du logo :

- **Format :** PNG
- **Fond :** Transparent
- **Couleurs :** Noir (#0B0B0B) et Or (#C59D3A)
- **Dimensions recommandées :** 400x100px (largeur x hauteur)
- **Résolution :** @2x pour affichage Retina (800x200px)

### Chemin d'accès :

```
blocksy-child/
└── assets/
    └── images/
        └── Logo-def_NOIR.png  ← Placer le logo ici
```

### Utilisation :

Le logo est utilisé dans :
- L'écran de chargement (Phase 1.1)
- Le header de l'interface principale (Phase 1.2)

### Alternative temporaire :

Si vous n'avez pas encore le logo, vous pouvez :
1. Créer un logo placeholder avec le texte "PianoMode"
2. Ou télécharger un logo temporaire depuis : https://pianomode.com
